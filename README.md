# AS grandMA3 Plugins

Lua plugins for grandMA3 lighting consoles (v2.3+) by AS.

## Installation

1. Download the `.zip` from the [Releases](https://github.com/asturde/asturde-ma3-plugins-public/releases) page
2. Extract the plugin folder to your MA3 plugins directory:
   - **macOS:** `~/MALightingTechnology/gma3_2.3.2/shared/resource/lib_plugins/`
   - **Windows:** `C:\ProgramData\MALightingTechnology\gma3_2.3.2\shared\resource\lib_plugins\`
3. In grandMA3: Menu > Plugins > Import > select the plugin

## Available Plugins

Check the [Releases](https://github.com/asturde/asturde-ma3-plugins-public/releases) page for the latest versions.
